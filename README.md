## CustomLintRules

自定义lint规则

gradle版本最低为4.2，4.1版本prepareLintJar任务会清除build/intermediates/lint/下的lint.jar，导致无效。

参考https://github.com/jzj1993/AndroidLint<br />
https://www.colabug.com/2109876.html


## 报告

执行`gradle lint`命令可以生成lint报告，查看各种issue。
